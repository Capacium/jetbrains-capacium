# JetBrains — Capacium Plugin

Search and install AI agent capabilities from the Capacium marketplace in any JetBrains IDE.

## Features

- Tool Window: Browse and search capabilities
- Trust badges: See trust state (discovered → signed)
- One-click install: Opens terminal with `cap install` command
- Status check: Verify connection to Capacium Exchange

## Install

From JetBrains Marketplace (coming soon) or build from source:

```bash
gradle buildPlugin
```
*(Gradle 8.7+ required. Run `gradle wrapper` once to generate `./gradlew`.)*

## IDE Support

- IntelliJ IDEA 2024.1+
- PyCharm 2024.1+
- WebStorm 2024.1+
- GoLand 2024.1+
- All IntelliJ Platform IDEs
